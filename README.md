# FitTrack

A fitness tracking web app for logging workouts, tracking macros, and monitoring progress.

## Live Demo
[https://fittrack-nine-gamma.vercel.app/](https://fittrack-nine-gamma.vercel.app/)

## Features
- User authentication (sign up / sign in)
- Log workouts across 4 splits (Upper A/B, Lower A/B)
- Track sets, reps, and weight for each exercise
- Log daily food intake and macros (calories, protein, carbs, fat)
- Set personal macro goals
- View training history and progress
- Suggested split based on your last workout
- Data persists across sessions

## Tech Stack
- React
- CSS (inline styles)
- localStorage for data persistence
- Deployed on Vercel

## Getting Started

Clone the repo and run locally:

```bash
git clone https://github.com/CARNAGE-TECH/fittrack.git
cd fittrack
npm install
npm start
```
